我们对这个进行分解：

宽度25%即一个盒子里平着放可以放四个。display：inline即在一条直线上。text-align:ceter是文本对齐；但是经过实验之后是图片居中，所以这个文本对齐是否是可以使整个图片（或者整个box）也居中；

从这里开始回溯，这四个小盒子（图标）在一个横线状的盒子里，这个横线盒子在一个盒子里有两个。宽度是在另一个盒子里撑满的

<div style="width: 100%">

在这个盒子里有四个宽度有25％的小盒子；

 <div style="width: 90%;margin: 0 auto;border-radius: 10px;background-color: white;padding-top: 17px;padding-bottom: 17px">

之后就是宽度90％，不占满整个项目，边框为圆框，背景色为白色，上下间距为17px

    <div style="width: 100%;;position: relative;top: -50px">

上一行代码是外层的边框。

总的来说

全部外边嵌套了一个大边框1

1中嵌入了4个边框，1、2、3、4

1、是图片

2、中套了各种图标其中带底色的是一个边框，不带底色的是最外层的边框，带底色中套了上下两个边框，上下两个边框中套了四个边框（图标），图标中套了两个边框分为图标和文字；

<template>
    <div class="tabbar-page">
        <van-tabbar>
            <van-tabbar-item to="/home">
                <span :class="active===0?fontActive:fontInActive">首页</span>
                <template #icon="props">
                    <img :src="active===0 ? icon.home.active : icon.home.inactive"/>
                </template>
            </van-tabbar-item>
            <van-tabbar-item to="/productsList">
                <span :class="active===1?fontActive:fontInActive">产品</span>
                <template #icon="props">
                    <img :src="active===1 ? icon.products.active : icon.products.inactive"/>
                </template>
            </van-tabbar-item>
            <van-tabbar-item to="/mine">
                <span :class="active===2?fontActive:fontInActive">我的</span>
                <template #icon="props">
                    <img :src="active===2 ? icon.mine.active : icon.mine.inactive"/>
                </template>
            </van-tabbar-item>
        </van-tabbar>
    </div>
</template>

<script>
    export default {
        name: 'MobileTabbar',
        props: {
            active: Number

        components: {},
